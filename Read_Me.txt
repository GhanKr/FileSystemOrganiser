File System Organiser
This project is about to create a file system organiser that
helps you to organise all your files and keep them to their respective folders.
Example->
if you have img, txt, doc ,app files randomly in a separate folder
then this code will help to create separate folders like images, documents and app and put 
all your files according to their extension types.
type -> Organise to organise all your files
type -> Tree to view all your folders and files in tree view mode
